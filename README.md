## ⚠️ Security Notice (Windows SmartScreen)

When running the `.exe`, Windows may show:

> “Windows protected your PC”

This happens because the app is **not code-signed (yet)**.

### ✔ How to proceed:

* Click **More info**
* Click **Run anyway**

💡 This is normal for independent/open-source tools.

## 🧩 How It Works

System Nexus Pro evaluates your system using:

* RAM thresholds (4GB / 8GB / 16GB)
* GPU VRAM detection
* Storage type (SSD vs HDD)

It then maps your specs to a **Gaming Capability Matrix**.
